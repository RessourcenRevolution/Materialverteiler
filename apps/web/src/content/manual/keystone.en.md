# Contentmanagement

Keystone is the content management system that allows you to manage content such
as navigation, pages, and footer.

## Accessing Keystone

The following page contains a link to Keystone: [Manage](/app/manage)

To get an account, someone with access must create a new account for you.

## User creation

1. Navigate to 'Users' in the main menu
2. Use the 'Create user' button to create a new user
3. Enter the user details
4. Save your changes

## Editing Navigation

1. Log in to Keystone
2. Navigate to "Navigation" in the main menu
3. Chose 'Main Navigation' (Main Navigation top-left) or 'Meta Navigation' (Main
   Navigation top-right)
4. Here you can add, edit, or remove navigation items
5. Save your changes

## Editing Pages

1. Log in to Keystone
2. Navigate to "Pages" in the main menu
3. Select the page you want to edit or create a new page
4. Use the text editor to edit content

## Adding Content Modules

Content modules are reusable content blocks that you can add to your pages:

1. Open the text editor of a page
2. Click on the "+" icon in the editor toolbar
3. Select the desired content module from the list
4. The module will be inserted into your content
5. Edit the module settings as needed

## Editing Images

Some content modules can include images. Images are managed on a separate page:

1. Navigate to "Images" in the main menu
2. Click 'Create Image'
3. Edit the fields and upload an image
4. Click 'Create Image'
5. The image is now available as a selection option in content modules

## Editing Footer

1. Log in to Keystone
2. Navigate to "Footer" in the main menu
3. Edit the footer content
4. Save your changes
